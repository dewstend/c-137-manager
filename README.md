# C-137 Manager

Para construir el código fuente, instalar node_modules.

    yarn
    // ó
    npm install
    
Instalar node_modules del cliente.

    cd ./client
    yarn
    // ó
    npm install

E inicializar y poblar la base de datos con la herramienta CLI.

    yarn initialize
    // ó
    npm run initialize
    
**Advertencia**: Inicializar limpiará datos existentes.
